# 内容选题助手 · 开源 Agent 技能

帮内容创作者（公众号 / 视频号 / 小红书 / 知乎 / 播客等）**发现、评估、策划选题**的 Agent 技能。把"拍脑袋选题"变成一套可复用的「发散 + 评估」流程。

> 内容提炼自 **刘正《选题的价值》** 培训 PPT，原作者保留署名权利。本技能依 MIT 许可开源，使用时请保留来源署名。

## 内容概览

1. **选题基础三问** —— 账号定位 / 人格化 / 为用户解决什么问题
2. **横纵发散法** —— 横向列板块 + 纵向细分阶段 → 选题池（单点 or 组合）
3. **10 大选题绝招** —— 追热点 / 反热点 / 找变化 / 想经历 / 有争议 / 总结套路 / 发现未知 / 用户痛点 / UGC / 换个形式
4. **选题价值 5 标准雷达评估** —— 热度 / 传播度 / 话题性 / 相关度 / 风险度
5. **追热点工具箱** —— 微博话题榜 / 百度风云榜 / 搜狗 / 微信指数 / 新榜等榜单链接
6. **标准工作流 + 输出清单格式 + 反模式**

## 安装

### WorkBuddy（本机已装位置）

技能已在：`~/.workbuddy/skills/内容选题助手/`

其他机器 / 账号安装：

```bash
# 方式一：git clone 到技能目录
git clone <仓库地址> ~/.workbuddy/skills/内容选题助手

# 方式二：手动复制目录
cp -r 内容选题助手 ~/.workbuddy/skills/
```

### 其他 Agent（跨平台兼容）

技能是标准 `SKILL.md` 格式，可放到任意 agent 的 skills 根目录：

| Agent | 技能目录 |
|-------|----------|
| WorkBuddy | `~/.workbuddy/skills/` |
| Claude Code | `~/.claude/skills/` |
| GitHub Copilot CLI | `~/.copilot/skills/` 或 `~/.agents/skills/` |
| Amp | `~/.config/agents/skills/` 或 `~/.config/amp/skills/` |
| 项目级 | 仓库内 `.claude/skills/` / `.agents/skills/` / `.github/skills/` |

## 使用

在 WorkBuddy 中直接说类似的话即可触发：

- 「用内容选题助手，帮我的运营类公众号想 10 个选题」
- 「评估一下这个选题值不值得做：＿＿」
- 「追一下今天的热点，给我能做的角度」
- 「我是新做母婴号，第一次选题该从哪入手？」

Agent 会读取 `SKILL.md`，按工作流给出选题清单（含角度、所用绝招、5 标准评分、风险标注）。
